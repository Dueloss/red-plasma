# Red Plasma Engine

Red Plasma is a cross-platform game engine built with C++ and Vulkan. 

## Project Goals
* **Separation**: The Engine and Editor are completely separate.
* **Modern Tech**: Built for Wayland and Linux first (Fedora).
* **Open Source**: A learning project for engine architecture.

## Tech Stack
* **Language**: C++
* **Graphics API**: Vulkan
* **Editor UI**: Qt6 (planned)
* **OS**: Fedora Linux (KDE)
